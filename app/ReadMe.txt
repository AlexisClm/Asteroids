Date : 27/11/2019

Auteur : CLEMENT Alexis

Description : Shoot 'em up (inspiré du jeu "Asteroids") où le but de la partie est de détruire les astéroïdes qui sont présents à l'écran


Contrôles :
	- Menu : Z = Navigation vers le haut
		 S = Navigation vers le bas
		 Entrée = Accès au jeu (ou quitter)

	- Jeu : Z = Avancer (Tips : pour reculer, faire un demi-tour avec le vaisseau et appuyer sur "Z")
		Q = Rotation vers la gauche
		D = Rotation vers la droite
		Espace = Tirer
		Echap = Accès aux options

	- Options : Z = Navigation vers le haut
		    S = Navigation vers le bas
		    Entrée = Retour au jeu (ou au menu ou quitter)
		    Echap = Retour au jeu

	- GameWin : Z = Navigation vers le haut
		    S = Navigation vers le bas
		    Entrée = Rejouer au jeu (ou quitter)

	- GameOver : Z = Navigation vers le haut
		     S = Navigation vers le bas
		     Entrée = Rejouer au jeu (ou quitter)
Bugs connus :
	- Les astéroïdes peuvent spawn instantanément sur le joueur
	- La collision du joueur avec l'asteroïde peut ne pas marcher si l'astéroïde est sur un des bords de l'écran
	- Les 2 astéroïdes peuvent se superposer quand il y a destruction d'un astéroïde

Fonctionnalités du jeu : Pour gagner, le joueur doit :
	- Ne pas se faire toucher par les astéroïdes (ne pas tomber à 0 HP)
	- Finir la 3ème wave qui comporte 4 gros astéroïdes
